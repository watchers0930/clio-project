# CLIO Role Action Matrix

## 역할 정의
- `admin`
  - 전사 운영 관리자
- `manager`
  - 부서 운영 관리자
- `user`
  - 일반 사용자

## 운영 해석
- `admin`은 시스템 운영 역할이다.
- `manager`는 부서 범위 운영 역할이지만 문서 수정 권한은 자동 확장되지 않는다.
- `user`는 일반 사용자지만 공유 문서에 대한 검토 참여는 가능하다.
- CLIO의 협업 확장은 역할보다 `명시적 공유`에 더 강하게 의존한다.
