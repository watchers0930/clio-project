# CLIO 메모 시스템 전면 고도화 기획서

## 1. 기능 개요
CLIO의 핵심 지식 자산인 메모 시스템을 단순 기록 도구를 넘어 **지식 간의 연결을 시각화하고 AI를 통해 새로운 인사이트를 도출하는 '지식 엔진'**으로 격상시킵니다. Obsidian 스타일의 동적 그래프 뷰와 AI 기반 아이디어 통합 생성, 다양한 포맷의 문서 출력을 지원합니다.

---

## 2. 사용자 스토리
1. **디자인 체감**: "더 넓은 여백과 깔끔한 폰트, 세련된 카드 디자인을 통해 정보를 한눈에 파악하고 싶다."
2. **그래프 시각화**: "메모들이 어떻게 연결되어 있는지 Obsidian처럼 직관적으로 보고, 주제별로 묶인 클러스터를 확인하고 싶다."
3. **AI 인사이트**: "연관된 메모들을 선택하면 AI가 이를 바탕으로 새로운 사업 아이디어나 기획안 초안을 작성해주면 좋겠다."
4. **결과물 출력**: "기획된 아이디어를 즉시 마크다운이나 공식 기획안(DOCX/HWPX) 템플릿에 맞춰 출력하고 싶다."
5. **실행 연결**: "도출된 아이디어에서 실행 가능한 할일(Todos)을 즉시 추출하여 업무로 등록하고 싶다."

## 3. 주요 기능 및 상세 사양

### 3.1 UI/UX 디자인 개선
- **여백 및 레이아웃**: 카드 간 간격(Gap) 최적화, 컨테이너 패딩 조정으로 가독성 향상.
- **폰트**: 시스템 폰트에서 브랜드 폰트(Paperlogy 등)로 통일.
- **카드 고도화**: 상태(Pinned), 생성일, 태그 가독성 개선 및 호버 애니메이션 추가.

### 3.2 Obsidian 스타일 그래프 클러스터링
- **실선/점선 연결**: 
  - **실선**: 제목 키워드 일치 또는 직접 연결.
  - **점선**: AI 임베딩(pgvector) 기반 의미론적 유사도(75% 이상).
- **클러스터링 시각화**:
  - 유사도 높은 메모 소그룹화(D3-Force Clustering).
  - 클러스터 영역에 배경 하이라이트(Hulls) 또는 버블 시각화.
- **자동 태깅/이름**:
  - 클러스터 내 메모 제목들을 분석하여 AI가 그룹 이름(예: "AI 법률 서비스", "신규 비즈니스") 자동 명명.

### 3.3 AI 아이디어 통합 생성
- **그룹 선택**: 그래프 뷰에서 드래그 또는 다중 클릭으로 메모 그룹 선택.
- **아이디어 생성**: 
  - 선택된 메모들의 맥락을 결합하여 GPT-4o 기반 신규 비즈니스/기획 아이디어 제안.
  - `POST /api/memos/idea` 고도화: 단순 요약을 넘어 구체적인 기획 프레임워크 적용.

### 3.4 다중 포맷 출력 (Export)
- **템플릿 선택**: 기획안, 보고서, 회의록 등 사전 정의된 템플릿 지원.
- **통합 엔드포인트**: `/api/generate`와 연동하여 Markdown, DOCX, HWPX, PDF 출력.
- **직접 작성 연동**: 생성된 아이디어를 기반으로 템플릿 내 플레이스홀더 자동 채우기.

### 3.5 할일(Todos) 연결
- **from-idea API 활용**: 아이디어 생성 결과물에서 실행 과제를 추출.
- **자동 등록**: `/api/todos/from-idea` 호출을 통해 우선순위와 함께 할일 목록에 즉시 반영.

## 4. 기술 요구사항

### 4.1 Frontend (React/Next.js)
- **Graph**: `react-force-graph-2d` 커스텀 페인팅 및 D3-force clustering 로직 강화.
- **Selection**: Canvas 내 영역 선택(Lasso selection style) 또는 멀티 셀렉트 UI.
- **State Management**: 선택된 메모 및 생성된 아이디어 상태 관리.

### 4.2 Backend (Node.js/Supabase)
- **API**:
  - `/api/memos/graph`: 클러스터링 데이터 및 그룹명 포함하도록 고도화.
  - `/api/memos/idea`: 스트리밍 지원 강화 및 출력 템플릿 연동.
  - `/api/todos/from-idea`: 할일 추출 프롬프트 고도화.
- **DB**: `memo_groups` 테이블 활용 및 캐싱 로직 최적화.

### 4.3 AI (OpenAI GPT-4o)
- **Prompt Engineering**: 클러스터 요약 및 아이디어 구체화 프롬프트 최적화.
- **Embedding**: pgvector를 통한 실시간 유사도 계산 성능 유지.

## 6. 기대 효과
- **업무 효율**: 파편화된 메모에서 기획안 초안까지의 과정을 5분 이내로 단축.
- **인사이트 발견**: 사용자 스스로 인식하지 못했던 정보 간의 연결고리를 시각적으로 발견.
- **지식 자산화**: 단순 기록물이 공식적인 문서 포맷으로 즉시 변환되어 자산화 가치 증대.
